# IoT-Hub-Node readme file

##Project Structure

```
-Controllers    //Controller, handling routing logic
-Models         //Data models, database interaction
-Services       //Business logic layer, handling complex business
-Middleware     //middleware, request preprocessing logic, such as permission verification
-Routes         //Route definition
-Config         //Configuration folder
-Utils          //utility functions or classes
-Public         //Static resource folder
-Views          //View folder
-Routes         //Routing folder
-Logs           //Log folder
-Tests          //Test folder
-App. js        //Entry file
```
